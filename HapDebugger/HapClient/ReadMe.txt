========================================================================
       CONSOLE APPLICATION : ConsoleEmbed
========================================================================

This program provides the "remote" portion of the HAP Remote Debugger. To
debug a program, open it with the DbgRemote program and press run - to debug
remotely, run this program on the remote machine and connect to it from the
DbgRemote program.

ConsoleEmbed.dsp
    This file (the project file) contains information at the project level and
    is used to build a single project or subproject. Other users can share the
    project (.dsp) file, but they should export the makefiles locally.

ConsoleEmbed.cpp
    This is the main application source file.


/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named ConsoleEmbed.pch and a precompiled types file named StdAfx.obj.
